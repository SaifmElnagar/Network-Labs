# 🌐 TCP/IP and OSI Models Lab (Packet Tracer)

## 📌 Lab Title
Investigating the TCP/IP and OSI Models in Action

---

## 🎯 Objectives

This lab aims to:

- Examine HTTP web traffic using Packet Tracer
- Understand how data moves through the TCP/IP protocol suite
- Visualize the relationship between TCP/IP and OSI models
- Explore encapsulation and protocol data units (PDUs)

---

## 📚 Background

This simulation activity provides a foundational understanding of how data travels across a network.

- Data is broken into smaller pieces called **PDUs (Protocol Data Units)**
- Each PDU is associated with a specific **layer** in the OSI and TCP/IP models
- Packet Tracer Simulation Mode allows step-by-step visualization of:
  - Packet movement
  - Encapsulation process
  - Layer-by-layer communication
